# Child-Navigation

**WordPress plugin for adding shortcode to list page and child or sibling pages.**

#### Description:
Creates a navigation block that lists the current page and any child pages, OR the current page and its sibling pages under the parent page.

## Setup

- Upload the folder to /wp-content/plugins/
- Activate from /wp-admin/plugins.php
- Use the shortcode [child_pages] where you want the navigation menu to appear on a page or template
